Social Media Project

Social media app on which user can create their own post including image video and words.

View Post All user log in/not log in
Create Post Only Log In User
Comment Post Only Log In User
Delete/Update Post Delete/Update Comment

Description of Envoy

Before log in users can view all posts but cannot comment on them. As a login user, users can create their own post with image/video words and comment on another person's post as well as view all content. For post and comment user created, it's easy to update and delete these content.
